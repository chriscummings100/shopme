# @chriscummings100/shopme

ShopMe command-line interface for controlling a live online grocery basket.

Start Chrome for a vendor login session:

```bash
npx -y @chriscummings100/shopme --vendor waitrose start
```

Then use commands such as:

```bash
npx -y @chriscummings100/shopme search "milk" --size 3
npx -y @chriscummings100/shopme cart
```
